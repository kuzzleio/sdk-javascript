# startQueuing

Starts the requests queuing. Works only in offline mode, and if the [autoQueue](/sdk/js/5/core-classes/kuzzle/#properties) option is set to `false`.

---

## Return Value

Returns the `Kuzzle` SDK object to allow chaining.
