# delete

Deletes a collection.

<br/>

```js
delete(index, collection, [options]);
```

<br/>

| Arguments    | Type              | Description     |
| ------------ | ----------------- | --------------- |
| `index`      | <pre>string</pre> | Index name      |
| `collection` | <pre>string</pre> | Collection name |


## Resolves

Resolves if the collection is successfully delete.
